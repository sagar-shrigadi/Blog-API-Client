# Blog Client Application

A Blog client built with React that fetches data from a self-coded RESTful [blog API](https://github.com/sagar-shrigadi/Blog-API) built with Express.js.

## 🔗 Live

- Live site can be found [here](https://blog-api-client.pages.dev/)
- The source code for blog-api (REST API) can be found [here](https://github.com/sagar-shrigadi/Blog-API).
- The source code for blog-api-cms (admin site) can be found [here](https://github.com/sagar-shrigadi/Blog-API-CMS).

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Styling:** Tailwind CSS

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git@github.com:sagar-shrigadi/Blog-API-Client.git
cd Blog-API-Client
```

### 2. Install dependencies

```bash
npm install
```

### 3. Configuration

- Create a `.env` file in the root directory and add the variable from .env.example file.

### 4. Start the Application

```bash
npm run dev
```
